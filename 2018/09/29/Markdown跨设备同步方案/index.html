Markdown跨设备同步方案
Tools Markdown
Tools
Publish Date:   2018-09-29
Word Count:   294
Read Times:   1 Min
Read Count:  

win + Linux + macOS + iOS ，PC端使用typora，手机端使用MWeb或Write.r，同步使用icloud；

PC端

typora安装请自行移步官网。

插入图片时如果使用绝对路径，移植到另一设备后，会出现找不到图片的问题。

typora设置图片路径：

  • 编辑》图片工具》当插入本地图片时》复制到文件夹
  • 编辑》图片工具》设置图片根目录

此后该.md文件内拖拽图片即可自动完成相对路径插入；如果想要跨设备，比如通过网盘或U盘移动文件夹，直接对主文件夹操作即可；此后使用Typora时直接打开整个主文件夹，一览无余。

附：在ubuntu终端安装typora

手机端

  • 手机是IOS系统，采用icloud同步方案；安卓系统请自行寻找其他同步方案，如坚果云

  • 传送门：IOS手机上面的Markdown软件试用对比

    推荐两款：

    1. MWeb
    2. 1Write
